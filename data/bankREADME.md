# Portuguese Bank Term Deposit Subscription Dataset

## Overview

This dataset is related to direct marketing campaigns conducted by a Portuguese banking institution, where campaigns were carried out via phone calls. In many cases, multiple contacts with the same client were necessary to determine whether they would subscribe to a bank term deposit product.

---

## Dataset Description

| Column | Description |
|--------|-------------|
| `age` | Age of the client. |
| `job` | Type of job the client holds. |
| `marital` | Marital status of the client. |
| `education` | Highest level of education attained by the client. |
| `default` | Whether the client has credit in default. |
| `housing` | Whether the client has a housing loan. |
| `loan` | Whether the client has a personal loan. |
| `contact` | Communication type used for the last contact. |
| `month` | Month of the last contact during the current campaign. |
| `day_of_week` | Day of the week of the last contact. |
| `duration` | Duration of the last contact in seconds. |
| `campaign` | Number of contacts performed during this campaign for this client. |
| `pdays` | Number of days since the client was last contacted from a previous campaign (-1 means not previously contacted). |
| `previous` | Number of contacts performed before this campaign for this client. |
| `poutcome` | Outcome of the previous marketing campaign. |
| `emp.var.rate` | Employment variation rate (quarterly social and economic indicator). |
| `cons.price.idx` | Consumer price index (monthly social and economic indicator). |
| `cons.conf.idx` | Consumer confidence index (monthly social and economic indicator). |
| `euribor3m` | Euribor 3-month rate (daily social and economic indicator). |
| `nr.employed` | Number of employees (quarterly social and economic indicator). |
| `y` | Whether the client subscribed to a term deposit. |
